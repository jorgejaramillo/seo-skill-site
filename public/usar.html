¿Cómo usar los SEO skills?

1. Clonas el repositorio en tu máquina local
2. Abres tu agente de IA (Claude Code, Cursor o V0)
3. Le pides que use un skill -> por ejemplo: "Valida las imágenes de esta web"
4. El agente ejecuta la auditoría y te entrega un reporte con lo que hay que corregir

No necesitas configurar nada. No necesitas saber programar. Solo necesitas un agente de IA compatible.



  • Claude Code — Agente de terminal de Anthropic
  • Cursor — Editor de código con IA integrada
  • V0 — Agente de desarrollo de Vercel
